# PROGRAM_ARVORE
### Descrição:
Este programa oferece funcionalidades básicas para manipulação de uma árvore binária em linguagem C. As operações incluem inserção, busca, impressão da árvore em pré-ordem, contagem de nós internos e identificação do menor valor na árvore.
## Modo de execução:
1- Navegue até o diretório onde está o arquivo teste.c do projeto na sua máquina;

2- Compile o programa usando GCC:
  ```sh
  gcc -o teste teste.c
  ```
3- Execute o programa:
  - No Linux/macOS, digite o seguinte comando no terminal:
    ```sh
    ./teste
    ```
  - No Windows (cmd.exe), digite o seguinte comando no terminal:
    ```sh
    teste.exe
    ```
  - No PowerShell, digite o seguinte comando no terminal:
    ```sh
    .\teste.exe
    ```
